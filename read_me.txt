You may begin running the docker environment by executing the python file: "docker_launcher.py" from the project's root directory.
The docker environment may also be shut down from the root directory by running the file "docker_shut_down.py"

To make your changes on your local machine reflect on Github, simply run the python file: "git_safe_push.py". This will prompt you for your name, and move your updates to the branch associated with your name.
If you want your local machine to reflect the changes on github, run the python file: "git_auto_pull.py"
